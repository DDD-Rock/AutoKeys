我现在需要写一个自动循环按键的软件。


窗口第一行是选择当前正在运行的软件，用于在每次按键之前，获取目标软件焦点再释放按键。

第二行是一个‘添加规则’按钮，按下后，创建一组规则

第三行开始是每组规则的规则列表：每条规则只支持设置单个按键，可以设置按键的延迟时间以及键值。按键的按下和抬起时间采用拟人化处理（随机微小波动，模拟真人操作）。多组规则各自独立倒计时，到时间后各自触发对应的按键（并行执行，互不影响）。每组规则需要在页面上显示当前的倒计时。每组规则之间用分隔线隔开，每组规则的右侧有一个删除按钮，可以删除该组规则。

每组规则添加后，有一个"选择按键"按钮，按下后弹出一个仿照键盘布局的弹窗，可以在弹窗中直观地点击选择目标按键。

最后一行是一个开始按钮，按下后，开始循环按键，再次按下停止。支持设置开始和停止的快捷键，默认是Ctrl+F1和Ctrl+F2，快捷键通过监听键盘输入来设置（点击设置按钮后按下目标组合键即可）。如果目标软件关闭，自动停止所有按键循环。软件自动保存规则到同目录下的配置文件中，下次打开自动加载。

每组规则可设置循环次数（无限循环 / 指定次数）。

窗口底部有状态栏，显示当前状态（运行中/已停止）以及已循环次数。

每组规则左侧有一个启用/禁用复选框，可以临时跳过该组规则而不用删除。
