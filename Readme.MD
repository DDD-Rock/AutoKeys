# AutoKeys (兔子按键精灵)

AutoKeys 是一款专为自动化循环按键需求设计的 Windows 桌面端软件，以“拟人化”、“高度定制化步调”及“多套路并发编排”为核心特点。支持将按键和等待操作极致分离，非常适合需要在指定应用程序中挂机、执行连招和复杂冷却战术的操作场景。

## 🌟 核心特性
- **目标窗口吸附**：精准捕获目标软件窗口句柄。每次触发按键前自动获取并切入焦点，当目标进程被关闭时自动暂停任务，贴心防越界。
- **高自由度的行动积木**：彻底解耦“按键”与“延时”！在一个套路规则中，你可以灵活平铺任意个纯净的 `[按键指令]` 和静默停驻的 `[⏳ 等待时长]`，像搭积木一样编排出极度复杂的爆发操作序列。
- **“爆发与冷却”双重延迟引擎**：
  - **整体触发延迟**：每一套大循环之间的长线冷却时间（例如等待10秒发一波连招大招）。
  - **步进延迟**：套路微操进行中内部的硬直等待时间（例如按下A以后，内部停顿3秒再按B）。
- **纯净拟人化保护**：底层封包按键行为时，强行注入了 50ms~150ms 的随机人手滞留力度以及抖动时间，让连续输入极富有生命和弹性，防止被死板判定。
- **虚拟全尺寸精美键盘**：抛弃简陋的 TextBox 读取，提供全键盘所见即所得的极简UI悬浮键盘交互页面，鼠标点选键位一目了然。
- **清晰透明的运行状态监视**：全局提供独立的多核计时通道。每个分组独立走线、互不干扰。界面实时刷新 `全局冷却倒数` ，并且引擎行至内部等待区间时会亮起 `淡蓝色指示灯高亮条`，同时独立递减内部时差。工作到哪一步，还要卡多久，一眼洞悉。

## 🚀 快速上手
1. 打开 `RabbitAutoKeys.exe`。
2. 顶栏下拉框选取想要进行自动化操作的宿主窗口标题程序。
3. 点击 **`[＋ 添加规则]`** 创建一个分组方案：
   - 填写 **循环次数**(填 0 永远循环) 和 爆发前的 **整体触发延迟(s)**。
   - 使用 **`[＋ 追加按键]`** 挂载瞬间点击的真按键（如选取 A）。
   - 使用 **`[＋ 追加间隔]`** 挂载发呆挂机秒数（如 3.0 秒）。
   *逻辑示意：(读秒等待全局冷却10秒) -> 瞬间按 E -> (卡在原地执行间隔3秒) -> 瞬间按 Q -> 结算，重新去循环初读秒10秒*
4. （可选）设定全局启停快捷键，默认为 `Ctrl+F1` (启动运行) 和 `Ctrl+F2` (急停)。
5. 按下顶部大大的 **`▶ 开始`** 按钮，或者键盘盲按 `Ctrl+F1`，尽情享受解放双手的体验！

## 🛠 架构说明与数据流
- 环境：仅要求 Windows 系统
- 运行框架：`.NET Framework 4.7.2`
- UI 层：WinForms 并搭载极具现代感的清爽极简样式与重绘控件
- 通信机制：底层封装了 Windows Native API (`user32.dll`) 直接进行原生硬件键盘事件广播与前台抓取操作。
- 持久化配置机制：引擎通过基于内存流的高性能无感自动存档，所有的宏规则都被序列化存放于下级目录的 `config.json` 里。随用随开，永不丢失。
